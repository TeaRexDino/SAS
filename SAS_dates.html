SAS Date/Time Formats Reference Chart

Date Formats

Format Description Example Output
DATEw. Writes date values in the form ddMMMyy or ddMMMyyyy DATE9. with value 22734 15MAR2022
DDMMYYw. Writes date values in the form dd/mm/yy or dd/mm/yyyy DDMMYY10. with value 22734 15/03/2022
DDMMYYBw. DD-MM-YY format with blank separator DDMMYYB10. with value 22734 15 03 2022
DDMMYYCw. DD:MM:YY format with colon separator DDMMYYC10. with value 22734 15:03:2022
DDMMYYDw. DD.MM.YY format with period separator DDMMDYD10. with value 22734 15.03.2022
DDMMYYPw. DD.MM.YY format with period separator (p) DDMMYYP10. with value 22734 15.03.2022
JULIANw. Writes date values as Julian dates in the form yyddd or yyyyddd JULIAN7. with value 22734 2022074
MMDDYYw. Writes date values in the form mm/dd/yy or mm/dd/yyyy MMDDYY10. with value 22734 03/15/2022
MONYYw. Writes date values in the form mmmyy or mmmyyyy MONYY7. with value 22734 MAR2022
NLDATEw. Writes date values in the form based on the locale NLDATE10. with value 22734 Depends on locale
WEEKDATEw. Writes date values in the form day-of-week, month-name dd, yyyy WEEKDATE28. with value 22734 Tuesday, March 15, 2022
WORDDATEw. Writes date values in the form month-name dd, yyyy WORDDATE18. with value 22734 March 15, 2022
YYMMDDw. Writes date values in the form yy/mm/dd or yyyy/mm/dd YYMMDD10. with value 22734 2022/03/15
E8601DAw. Writes date values in the ISO 8601 extended notation yyyy-mm-dd E8601DA10. with value 22734 2022-03-15

Time Formats

Format Description Example Output
HHMMw.d Writes time values as hours and minutes in the form hh:mm HHMM5. with value 43200 (12 hours) 12:00
HOURw.d Writes time values as hours and decimal fractions HOUR5.1 with value 43200 (12 hours) 12.0
MMSSw.d Writes time values as minutes and seconds MMSS5. with value 43200 (12 hours) 00:00 (mins and secs portion only)
TIMEw.d Writes time values in the form hh:mm:ss.ss TIME8. with value 43200 (12 hours) 12:00:00
TODw.d Writes time values using AM/PM notation TOD8. with value 43200 (12 hours) 12:00 PM
NLTIMEw. Time value based on locale-specific settings NLTIME8. with value 43200 (12 hours) Depends on locale settings

Datetime Formats

Format Description Example Output
DATETIMEw. Datetime value (date and time) DATETIME20. with value 1970000000 22MAR2022:17:20:00
DATETIME18.w ISO 8601 datetime format DATETIME18. with value 1970000000 2022-03-22T17:20:00
DTDATEw. Date part of a datetime value DTDATE9. with value 1970000000 22MAR2022
DTMONYYw. Month and year part of a datetime value DTMONYY7. with value 1970000000 MAR2022
DTWKDATXw. Day of week and date part of a datetime value with international format DTWKDATX29. with value 1970000000 Tuesday, 22 March 2022
DTYEARw. Year part of a datetime value DTYEAR4. with value 1970000000 2022
DTYYQCw. Year and quarter part of a datetime value DTYYQC7. with value 1970000000 2022:Q1
E8601DTw. ISO 8601 extended format datetime E8601DT19. with value 1970000000 2022-03-22T17:20:00
B8601DTw. ISO 8601 basic format datetime B8601DT16. with value 1970000000 20220322T172000

Notes:

  • w: Represents the total width of the formatted value
  • SAS date values are stored as the number of days since January 1, 1960
  • SAS time values are stored as the number of seconds since midnight
  • SAS datetime values are stored as the number of seconds since January 1, 1960

This chart includes date and time formats from the SAS documentation. For the most up-to-date and complete information, please refer to the official SAS documentation.
